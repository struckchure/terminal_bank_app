# Terminal Bank Application

## Usage
1. Clone the repo
    ```bash
    git clone https://github.com/Olayiwolaaa/terminal_bank_app.git
    ```
2. Install dependency and run app.py
    ```bash
    pip install -r requirements.txt
    python app.py
    ```
